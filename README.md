# IT Support Labs

Hands-on troubleshooting labs built to strengthen practical, entry-level IT support skills.

Each lab simulates a common helpdesk or desktop support scenario using *VirtualBox, primarily in **Windows 10* environments, with selected infrastructure labs using *Windows Server 2022*. Every lab follows a structured troubleshooting workflow and includes documentation, screenshots, and real-world relevance.

This portfolio is designed to demonstrate practical proof of work for roles such as:
- IT Support
- ICT Support
- Helpdesk / Service Desk
- Desktop Support
- Junior Infrastructure Support
- 
## Why These Labs Matter
Many entry-level IT support interviews focus less on theory and more on whether you can troubleshoot common user issues in a structured way.

These labs were built to improve practical confidence in:
- Windows troubleshooting
- Basic network troubleshooting
- User and device support
- Windows services and system tools
- Active Directory basics
- Remote support scenarios
- Performance and storage issues
- Log-based troubleshooting with Event Viewer

## Tech Stack Used
- VirtualBox
- Windows 10
- Windows Server 2022
- Command Prompt (ipconfig, netsh)
- Windows Services (services.msc)
- Event Viewer (eventvwr.msc)
- Active Directory Users and Computers
- Group Policy
- AnyDesk
- Markdown for documentation
