# IT Support Labs

Hands-on troubleshooting labs built to develop real entry-level IT support skills. Each lab simulates common helpdesk tickets using VirtualBox for Windows 10, follows the OSI model bottom-up, and includes full documentation, screenshots and real-world relevance.

Perfect for CCNA-level interviews and junior helpdesk/desktop support roles at telcos, banks, and MSPs in South Africa.

## Why These Labs Matter
- 30–40% of entry-level helpdesk tickets in SA are connectivity or basic configuration issues (Layer 1–3).
- Builds the exact troubleshooting skills recruiters test in interviews (e.g., “Walk me through fixing no internet”).

## Tech Stack Used
- VirtualBox (Windows 10 VMs)
- Cmd commands (ipconfig, netsh)
- Markdown for documentation
