# IT Support & Infrastructure Labs

Hands-on troubleshooting and infrastructure labs built to develop practical skills for entry-level IT support, ICT support, and junior infrastructure roles.

This repository is structured in phases to show progression from desktop support fundamentals into server infrastructure, Active Directory, networking, and security.
