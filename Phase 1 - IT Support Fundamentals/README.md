# Phase 1 - IT Support Fundamentals

This phase focuses on practical entry-level IT support and helpdesk troubleshooting using Windows 10 in VirtualBox.

These labs simulate common real-world support tickets such as connectivity issues, printer problems, service failures, user account lockouts, and event log investigation.

The goal of this phase is to build strong troubleshooting fundamentals before progressing into Windows Server, Active Directory, and infrastructure support.
